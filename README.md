# UNDER DEVELOPMENT

## Installation

With [npm](https://www.npmjs.com/package/cccompiler):

```sh
$ sudo npm install -g cccompiler
```

## Source code

Source repository is on [github](https://github.com/iccicci/ccc).

## Usage

### From shell

```sh
$ ccc [options] file
```

### As module

```javascript
var ccc = require("cccompiler");
var res = ccc(filename, options);
```
